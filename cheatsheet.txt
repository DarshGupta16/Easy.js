							--------Easy.js Cheatsheet------

select(elem) - Selects whatever is passed in via id or class entered. Eg: select('#mySelectElem') or select('.mySelectElem')

selectAll(elem) - Works like querySelectorAll in JS

getInputValue(elem) - Gets value of input via input id or class entered

onClick(elem, functionToExecute) - Adds a click event listener to the specified element passed in via id or class in the first argument and executes the second argument (the function to be executed)

onChange(elem, functionToExecute) - Does the same thing as onClick, but adds a change event listener to the element passed in.

changeHTML(elem, text) - Changes the HTML of the element passed in.
